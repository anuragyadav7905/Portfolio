# Anurag Singh Yadav — Portfolio

> "Ship fast, iterate faster."

🌐 **Live Site:** [portfolio-steel-psi-31.vercel.app](https://portfolio-steel-psi-31.vercel.app/)

---

## Tech Stack

- **React** + **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**

## Sections

- **Hero** — Animated background, role badges, social links
- **About** — Bio, core strengths, domain overview
- **Projects** — 9 projects with tags and GitHub links
- **Skills** — 8 categorized skill groups
- **Contact** — GitHub, LinkedIn, Email

## Run Locally

```bash
git clone https://github.com/anuragyadav7905/Portfolio.git
cd Portfolio
npm install
npm run dev
```
